1) Перейти в каталог проекта.
    cd 'Макушинская Д.В./lab06'
2) Перейти в каталог с функцией генерации файла и выполнить сборку программы.
    cd gen_file
    make
3) Выполнить генерацию файла. (Пример значения value_str - 8192)
    ./gen value_str
4) Перейти в каталог с функцией сортировки и выполнить сборку программы.
    cd ../sort
    make
5) Отсортировать созданный файл. (Примеры значений value_str - 8192, value_blocks - 32, value_thr - 8, path -  "../gen.txt")
   ./sort  value_str value_blocks value_thr path
6) Перейти в каталог с функцией вывода содержимого файла и собрать программу.
   cd ../print_file
   make
7) Вывести содержимое отсортированного файла на экран. (Пример значения value_str - 8192)
    ./print_file valueOfStruct
